# How It Works Section - Image Assets

This folder contains the images displayed in the "How It Works" section of the landing page.

## Required Images

### Step 01 - Upload Your Photo (2 images side-by-side)
- `step1_a.jpg` - First image (e.g., front-facing photo example)
- `step1_b.jpg` - Second image (e.g., side-facing photo example)

### Step 02 - Get Your Analysis (1 image)
- `step2.jpg` - Analysis visualization (e.g., facial analysis screenshot)

### Step 03 - Follow Your Plan (1 image)
- `step3.jpg` - Plan visualization (e.g., personalized recommendations screenshot)

### Step 04 - Track Your Progress (1 image)
- `step4.jpg` - Progress tracking visualization (e.g., before/after comparison or chart)

## Image Specifications

- **Dimensions**: Recommended aspect ratio 16:9 or similar
- **Size**: Optimized for web (under 500KB per image)
- **Format**: JPG or PNG
- **Style**: Should match the dark/luxury theme of the landing page
- **Quality**: High-quality, professional images

## Fallback Behavior

If images are missing or fail to load, a gradient fallback will be displayed automatically.
No action needed - the component handles errors gracefully.
